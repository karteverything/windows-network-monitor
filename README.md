# Windows Network Monitor

A lightweight Python tool that displays real-time upload and download speeds directly from the Windows system tray.

## Features

* Real-time network speed monitoring
* System tray integration
* Lightweight and minimal UI

## Tech Stack

* Python
* psutil
* pystray
* Pillow

## Setup

```bash
pip install -r requirements.txt
python main.py
```
